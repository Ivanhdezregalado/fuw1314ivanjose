Cabecera

Preparación MV

Primero que nada establecemos IP's estáticas a nuestras MV's.



Instalar italc master

  1. Empezamos instalando el italc-master en una MV que será la que llevará el control de las demás.



  2. Luego tenemos que ejecutar el siguiente comando para que se creen unos ficheros llamados keys para usarlos más adelante.



  3. Ahora creamos el grupo italc y añadimos nuestro usuario al grupo.



  4. Con el siguiente comando damos permisos a los usuarios del grupo italc para que tengan acceso a las keys.



  5. Verificamos que el servicio esta iniciado. Si no está iniciado ejecutar el comando "/usr/bin/ica &" para iniciarlo.



  6. Para terminar al final del fichero ".profile" del usuario que se encuentra en su home añadimos la siguiente linea al final para que se inicie automaticamente el servicio italc al inicio del sistema: "/usr/bin/ica &".



Instalar italc cliente en Windows

  1. Primero nos descargamos el italc cliente de alguna página web segura y procedemos con la instalación.



  2. Para llevar la instalación con éxito debemos copiar la "key" del public del master para que el instalador de Windows nos la coja. Para ello cambiamos el nombre de la "key" por "italc_dsa_key.pub"





  3. Finalmente, comprobamos que está iniciado el servicio.



  4. Desactivamos el firewall por si hubieran problemas de conexión entre las MV. Y le proporcionamos al programa italc cliente permisos.





Instalar italc cliente en Linux

  1. Empezamos la instalación del cliente el una MV debian.



  2. Luego creamos el grupo italc y añadimos el usuario al grupo.



  3. Ahora pasamos la "key" del master al cliente.



  4. Le ponemos permisos al grupo italc para acceder a las keys.



  5. Para terminar al final del fichero ".profile" del usuario que se encuentra en su home añadimos la siguiente linea al final para que se inicie automaticamente el servicio italc al inicio del sistema: "/usr/bin/ica &".



Comprobación

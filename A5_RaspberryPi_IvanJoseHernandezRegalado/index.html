Descarga del sistema | Botear la tarjeta SD | Raspberry | Configuración del SO

Cabecera

Descarga del sistema

  1. Vamos a la página oficial de la Raspberry para descargar un sistema operativo.



  2. Luego descomprimimos el zip para tener la imagen a mano mas tarde.

Botear la tarjeta SD

  1. Nos descargamos el programa Win32DiskImager para pasar la iso a la SD y que la Raspberry la ejecute.

  2. Luego ejecutamos el programa y ponemos la SD en el adaptador SD-USB para que el programa nos convierta la SD en un boteable con la iso dentro.



NOTA: cuidado con la SD no todas son compatibles para instalar en la Raspberry, aqui dejo un link con las SD compatibles y no compatibles para evitar problemas. SD's ompatibles y no compatibles

Raspberry

  1. Empezamos poniendo la Raspberry con su carcasa.



  2. Luego la conectamos a la tele mediante el cable HDMI, la conectamos a la corriente, le ponemos la SD, un teclado y ratón USB.



Configuración del SO

  1. Para empezar entramos como root y la contraseña root para configurar el sistema operativo y ejecutamos el comando "mv /boot/config.txt /boot/config.old" y "reboot".



  2. Luego le cambiamos la contraseña al root.



  3. Buscamos nuestra zona horaria y hacemos un link simbolico en /etc/localtime.



  4. Miramos como está organizado nuestro sistema.



  5. Creamos pacman que es nuestro actualizador con los siguientes comandos.



  6. Creamos un usuario llamado pi con contraseña profesor.




  7. Por último miramos la MAC de nuestra Raspberry y la apuntamos en la carcasa.
